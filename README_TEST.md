# 策略参数组合自动化测试

## 概述

这个脚本用于自动测试不同参数组合下的策略表现，帮助找到最优的策略配置。

## 测试参数

### 权重参数（步长0.05，总和必须=1）
- **市值权重** (mvWeight): 0.00 - 1.00
- **股息率权重** (dvWeight): 0.00 - 1.00
- **质量因子权重** (qualityWeight): 0.00 - 1.00

### 其他参数
- **单只股票最大权重** (maxWeight): 0.05 (5%), 0.10 (10%)
- **质量因子类型** (qualityFactorType): 
  - `pe_pb`: PE+PB综合（默认）
  - `pe`: 仅市盈率PE
  - `pb`: 仅市净率PB
  - `roe`: ROE（净资产收益率）

### 测试期间
- 开始日期: 2018-11-30
- 结束日期: 2025-12-15

## 预计测试用例数

- 权重组合数: 231个（满足总和=1的所有组合）
- 最大权重选项: 2个
- 质量因子类型: 4个
- **总计**: 231 × 2 × 4 = **1,848个测试用例**

## 使用方法

### 1. 确保服务器运行
```bash
npm start
```

### 2. 运行测试脚本
```bash
node test_strategy_combinations.js
```

### 3. 查看结果
测试完成后，会在 `docs` 目录下生成两个文件：
- `strategy_test_results.md`: Markdown格式的测试报告
- `strategy_test_results.json`: JSON格式的原始数据

## 输出报告内容

### 测试结果汇总表
包含每个测试用例的详细指标：
- 参数配置（市值/股息率/质量因子权重，质量因子类型，最大权重）
- 年化收益率
- 波动率
- 夏普比率
- 最大回撤
- 超额收益（相对指数）
- 跟踪误差
- 测试状态

### 最佳策略分析
自动识别并展示：
- **最高年化收益率**的策略配置
- **最高夏普比率**的策略配置
- **最小最大回撤**的策略配置

## 注意事项

1. **测试时间**: 每个测试用例需要约30-60秒，总测试时间约15-30小时
2. **中间结果**: 每完成10个测试会自动保存中间结果到 `strategy_test_results_temp.md`
3. **错误处理**: 如果某个测试失败，会记录错误信息并继续下一个测试
4. **超时设置**: 每个API调用超时时间为5分钟

## 自定义配置

如需修改测试参数，编辑 `test_strategy_combinations.js` 文件中的配置：

```javascript
const START_DATE = '20181130';        // 开始日期
const END_DATE = '20251215';          // 结束日期
const WEIGHT_STEP = 0.05;             // 权重步长
const QUALITY_FACTOR_TYPES = ['pe_pb', 'pe', 'pb', 'roe'];  // 质量因子类型
const MAX_WEIGHTS = [0.05, 0.10];     // 最大权重选项
```

## 故障排除

### 问题：连接超时
**解决**: 检查服务器是否运行，端口是否正确（默认3001）

### 问题：内存不足
**解决**: 减少并发测试数量，或增加Node.js内存限制
```bash
node --max-old-space-size=4096 test_strategy_combinations.js
```

### 问题：测试中断
**解决**: 查看 `strategy_test_results_temp.md` 中的中间结果，可以修改脚本跳过已完成的测试
