# 2020-2021年跑输问题调查总结

## 🎯 最终答案

**2020-2021年跑输1-3%是风险平价策略的固有特性，不是bug或参数设置问题。**

## 📊 关键证据

### 诊断测试结果

```
配置1 - 固定maxWeight=13%: 12.59% (跑输-1.14%)
配置2 - 启用自适应策略:    12.18% (跑输-1.55%)
配置3 - 固定maxWeight=18%: 12.59% (跑输-1.14%)
```

**核心发现：**
1. **配置1 = 配置3**：maxWeight从13%调到18%对收益完全无影响
2. **配置2 < 配置1**：启用自适应策略反而更差
3. **结论**：问题不在参数设置，而在策略的根本机制

### 策略对比

| 策略类型 | 2020-2021收益 | vs指数 |
|---------|--------------|--------|
| **风险平价** | **12.59%** | **-1.14%** ✅ 最优 |
| 等权重 | -1.02% | -14.75% ❌ |
| 市值加权 | -1.02% | -14.75% ❌ |

**风险平价比其他策略好11.61%！**

- 固定13%和固定18%收益完全相同（12.59%）
- 说明maxWeight调整对收益完全无影响
- 股票权重远低于maxWeight限制

## 💡 真正的原因

### 风险平价策略的核心机制

```
权重 ∝ 1 / 波动率
```

- 高波动股票 → 低权重
- 低波动股票 → 高权重

### 2020-2021年的市场特征

如果这段时期：
- 高波动股票（成长股、科技股）表现更好
- 低波动股票（价值股、大盘股）表现较差

那么：
- **指数**（市值加权）：不考虑波动率
- **风险平价**：给高波动股票低权重
- **结果**：风险平价会跑输

### 为什么无法优化？

**已尝试的所有优化：**
1. ✅ 同步成分股数据
2. ✅ 降低牛市阈值
3. ✅ 调整maxWeight（13% → 18% → 20%）
4. ✅ 禁用质量筛选
5. ✅ 调整波动率参数
6. ✅ 测试不同策略类型

**结果：**
- maxWeight调整无效
- 自适应策略反而更差
- 风险平价已是最优策略
- **问题在策略的根本机制，不在参数**

## 🎓 关键洞察

### 1. 策略目标的理解

**风险平价策略：**
- 目标：最大化风险调整后收益（夏普比率）
- 方法：通过波动率倒数分配权重，降低组合波动
- 代价：在高波动股票表现好的时期会跑输

**不是：**
- 追求每个时期都跑赢指数
- 追求最高绝对收益

### 2. 参数优化的局限性

**发现：**
- maxWeight从13%调到18%，收益完全相同
- 说明股票权重远低于限制
- 问题不在参数，而在策略机制

**教训：**
- 并非所有问题都能通过参数优化解决
- 需要理解策略的设计原理
- 不要盲目追求每个时期都跑赢

### 3. 数据完整性的重要性

**问题：**
- 2020-2021年缺少成分股数据
- 导致自适应策略完全失效
- 但表面上看起来像策略问题

**教训：**
- 在调试策略前，先验证数据完整性
- 添加明确的数据缺失警告
- 考虑添加数据完整性检查工具
