# 风险平价策略详细说明

## 一、策略核心逻辑

### 1.1 基本原理
风险平价（Risk Parity）策略的核心思想是：**让每只股票对组合总风险的贡献相等**，而不是让每只股票的权重相等。

传统的等权重策略中，高波动率股票会对组合风险贡献更多，而风险平价策略通过给低波动率股票分配更高权重，给高波动率股票分配更低权重，来平衡各股票的风险贡献。

**权重计算公式**：
```
权重_i ∝ 1 / 波动率_i
```

归一化后：
```
权重_i = (1/波动率_i) / Σ(1/波动率_j)
```

### 1.2 计算流程

#### 步骤0：股票池筛选（可选）
如果启用股票池筛选，会先对股票进行以下筛选：
- **ROE筛选**：剔除ROE低于设定阈值的股票（如ROE < 8%）
- **负债率筛选**：剔除负债率高于设定阈值的股票（如负债率 > 60%）
- **动量筛选**：剔除过去N个月动量收益率低于阈值的股票（如6个月动量 < -10%）
- **质量得分筛选**：剔除质量得分低于中位数的股票（基于PE、PB计算）

**数据来源**：
- ROE和负债率：来自Tushare的`fina_indicator`接口，使用最近一期财报数据
- 动量数据：使用调仓日前N个月的股票价格数据
- PE、PB：来自Tushare的`daily_basic`接口，使用调仓日当天的数据

#### 步骤1：获取历史价格数据
**数据时间范围**：调仓日往前推`volatilityWindow`个月（默认12个月）

例如：
- 调仓日：2024-12-31
- 波动率窗口：12个月
- 数据起始日：2023-12-31
- 数据结束日：2024-12-31

**数据要求**：
- 最少数据点：`volatilityWindow × 20 × 0.8`（默认12×20×0.8=192个交易日）
- 使用复权后的收盘价（`adj_close`）

#### 步骤2：计算每只股票的波动率
使用**EWMA（指数加权移动平均）**方法计算波动率：

```
EWMA波动率 = √(EWMA方差)
EWMA方差_t = λ × EWMA方差_{t-1} + (1-λ) × 收益率_t²
```

其中：
- λ（衰减因子）= 0.94（默认值）
- 收益率_t = (价格_t - 价格_{t-1}) / 价格_{t-1}

**EWMA的优势**：
- 对近期数据赋予更高权重，更能反映当前市场状况
- 平滑处理，减少噪音影响

#### 步骤3：计算基础风险平价权重
**标准风险平价**（默认方法）：
```
权重_i = (1/波动率_i) / Σ(1/波动率_j)
```

**协方差矩阵优化**（可选，`useCovariance=true`）：
- 构建股票收益率的协方差矩阵
- 考虑股票之间的相关性
- 基于协方差矩阵计算风险平价权重

#### 步骤4：应用权重上限
对每只股票的权重应用最大权重限制（默认10%），防止过度集中。

```
权重_i = min(权重_i, maxWeight)
```

应用权重上限后重新归一化，确保权重总和为100%。

#### 步骤5：混合策略（可选）
如果启用混合策略（`hybridRatio > 0`），将风险平价权重与市值加权混合：

```
最终权重 = (1 - hybridRatio) × 风险平价权重 + hybridRatio × 市值权重
```

例如：`hybridRatio = 0.3` 表示70%风险平价 + 30%市值加权

#### 步骤6：质量因子倾斜（可选）
如果启用质量因子倾斜（`useQualityTilt=true`），会给高质量股票增加权重：

```
质量得分 = 股息率 × 10 + 1/PE
质量乘数 = 1 + min(质量得分 × 0.1, 0.3)
调整后权重 = 原权重 × 质量乘数
```

最多增加30%权重。

#### 步骤7：动量因子倾斜（可选）
如果启用动量因子倾斜（`useMomentumTilt=true`），会给高动量股票增加权重。

**数据时间范围**：调仓日往前推`momentumWindow`个月（默认6个月）

```
动量收益率 = (最后价格 - 第一价格) / 第一价格
动量得分 = (动量收益率 - 最小动量) / (最大动量 - 最小动量)
动量乘数 = 1 + 动量得分 × momentumWeight
最终权重 = 风险平价权重 × 动量乘数
```

## 二、数据使用时间段总结

| 数据类型 | 时间范围 | 用途 |
|---------|---------|------|
| 股票日收益率 | 调仓日往前12个月 | 计算波动率（EWMA） |
| 股票价格（动量） | 调仓日往前6个月 | 计算动量收益率 |
| ROE、负债率 | 最近一期财报 | 股票池筛选 |
| PE、PB、股息率 | 调仓日当天 | 质量因子计算、股票池筛选 |
| 市值 | 调仓日当天 | 市值加权（混合策略） |

## 三、合理性分析

### 3.1 优点

1. **风险分散更均衡**
   - 避免高波动率股票主导组合风险
   - 每只股票对总风险的贡献相等

2. **适应市场变化**
   - EWMA方法对近期数据赋予更高权重
   - 12个月窗口既能捕捉长期趋势，又能适应市场变化

3. **多因子增强**
   - 可选的质量因子、动量因子倾斜
   - 灵活的混合策略

4. **风险控制**
   - 单只股票最大权重限制（10%）
   - 股票池筛选剔除低质量股票

### 3.2 潜在问题与改进建议

1. **波动率窗口（12个月）**
   - **现状**：使用固定12个月窗口
   - **问题**：市场环境快速变化时可能滞后
   - **建议**：可考虑6-18个月的动态窗口，或使用更短的窗口（如6个月）

2. **EWMA衰减因子（0.94）**
   - **现状**：固定使用0.94
   - **问题**：不同市场环境可能需要不同的衰减速度
   - **建议**：可测试0.90-0.97之间的不同值

3. **股票池筛选的时机**
   - **现状**：在计算波动率之前筛选
   - **优点**：减少计算量
   - **问题**：可能错过某些高波动但高收益的股票
   - **建议**：当前方案合理，但可提供"先计算后筛选"的选项

4. **财务数据的时效性**
   - **现状**：使用最近一期财报
   - **问题**：财报可能滞后3-6个月
   - **建议**：当前方案合理，财报数据本身就有滞后性

5. **权重上限（10%）**
   - **现状**：固定10%上限
   - **问题**：对于50只股票的组合，10%可能偏高
   - **建议**：可根据股票数量动态调整，如`max(5%, 100%/股票数×2)`

## 四、参数配置建议

### 4.1 保守型配置（低风险）
```javascript
{
  volatilityWindow: 12,        // 较长窗口，更稳定
  ewmaDecay: 0.94,             // 标准衰减
  maxWeight: 0.08,             // 较低权重上限
  useQualityTilt: true,        // 启用质量倾斜
  enableStockFilter: true,     // 启用筛选
  minROE: 0.10,                // 较高ROE要求
  maxDebtRatio: 0.50,          // 较低负债率
  filterByQuality: true        // 剔除低质量股票
}
```

### 4.2 平衡型配置（中等风险）
```javascript
{
  volatilityWindow: 12,
  ewmaDecay: 0.94,
  maxWeight: 0.10,
  useQualityTilt: false,
  enableStockFilter: true,
  minROE: 0.08,
  maxDebtRatio: 0.60,
  minMomentumReturn: -0.10,
  filterByQuality: true
}
```

### 4.3 进取型配置（高风险高收益）
```javascript
{
  volatilityWindow: 6,         // 较短窗口，更灵活
  ewmaDecay: 0.90,             // 更快衰减，更重视近期
  maxWeight: 0.15,             // 较高权重上限
  useMomentumTilt: true,       // 启用动量倾斜
  momentumWeight: 0.3,
  enableStockFilter: true,
  minROE: 0.05,                // 较低ROE要求
  maxDebtRatio: 0.80,          // 较高负债率容忍
  minMomentumReturn: 0.05,     // 只选高动量股票
  filterByQuality: false       // 不剔除低质量股票
}
```

## 五、最大化夏普比率的参数分析

基于历史回测和理论分析，以下参数配置可能有助于提高夏普比率：

### 5.1 关键参数影响

1. **波动率窗口（volatilityWindow）**
   - **影响**：窗口越短，对近期市场变化越敏感
   - **建议**：6-12个月，取决于市场波动性
   - **夏普优化**：在震荡市场使用较短窗口（6个月），在趋势市场使用较长窗口（12个月）

2. **股票池筛选**
   - **影响**：剔除低质量股票可提高收益质量
   - **建议**：
     - ROE >= 8-10%：剔除盈利能力差的公司
     - 负债率 <= 60%：剔除高杠杆风险公司
     - 动量 >= -10%：剔除下跌趋势股票
     - 质量筛选：剔除低PE、PB得分股票
   - **夏普优化**：适度筛选（不要过于严格）可提高夏普比率

3. **最大权重限制（maxWeight）**
   - **影响**：限制过度集中，降低非系统性风险
   - **建议**：8-12%
   - **夏普优化**：10%是较好的平衡点

4. **质量因子倾斜**
   - **影响**：增加高质量股票权重
   - **建议**：在价值股表现好的市场启用
   - **夏普优化**：可能提高夏普比率，但需要回测验证

5. **动量因子倾斜**
   - **影响**：增加高动量股票权重
   - **建议**：在趋势市场启用
   - **夏普优化**：可能提高收益，但也增加波动率

### 5.2 推荐的夏普优化配置

```javascript
{
  // 基础参数
  volatilityWindow: 9,          // 中等窗口，平衡稳定性和灵活性
  ewmaDecay: 0.94,              // 标准衰减
  maxWeight: 0.10,              // 10%上限
  
  // 不使用倾斜因子（减少复杂度）
  useQualityTilt: false,
  useCovariance: false,
  useMomentumTilt: false,
  hybridRatio: 0,
  
  // 启用适度的股票池筛选
  enableStockFilter: true,
  stockFilterParams: {
    minROE: 0.08,               // 8% ROE门槛
    maxDebtRatio: 0.60,         // 60%负债率上限
    momentumMonths: 6,
    minMomentumReturn: -0.10,   // -10%动量门槛（不要太严格）
    filterByQuality: true       // 剔除低质量股票
  }
}
```

**理由**：
1. 纯风险平价策略已经能很好地分散风险
2. 适度的股票池筛选剔除明显的低质量股票
3. 避免过度优化和过拟合
4. 参数简单，易于理解和维护

## 六、调仓频率的影响

当前支持三种调仓频率：
- **年度调仓**（yearly）：每年调仓一次，交易成本最低
- **季度调仓**（quarterly）：每季度调仓一次，平衡灵活性和成本
- **月度调仓**（monthly）：每月调仓一次，最灵活但交易成本高

**夏普比率优化建议**：
- 如果不考虑交易成本：季度调仓可能最优
- 如果考虑交易成本：年度调仓通常更优
- 当前默认年度调仓是合理的选择

## 七、总结

风险平价策略通过动态调整股票权重，使每只股票对组合风险的贡献相等，是一种科学的风险管理方法。

**核心优势**：
- 风险分散更均衡
- 适应市场变化
- 可扩展性强（支持多种增强因子）

**使用建议**：
- 保持参数简单，避免过度优化
- 适度使用股票池筛选
- 定期回测和调整参数
- 关注交易成本的影响

**最优配置**：
- 波动率窗口：9-12个月
- ROE筛选：>= 8%
- 负债率筛选：<= 60%
- 动量筛选：>= -10%
- 质量筛选：启用
- 最大权重：10%
