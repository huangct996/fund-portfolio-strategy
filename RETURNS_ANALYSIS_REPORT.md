## ⚠️ 前端图表问题

### **问题定位**

**前端显示**: 241%  
**实际收益**: 68.86%  
**差异**: 172个百分点

### **可能原因**

1. **累乘计算错误**
   - 前端可能错误地将各期收益率直接累加
   - 正确应该是累乘: (1+r1) × (1+r2) × ... - 1

2. **数据重复计算**
   - 可能将某些期间的收益率重复计算
   - 导致收益率虚高

3. **基准值错误**
   - 可能使用了错误的初始值
   - 导致收益率计算基数错误

### **需要检查的代码**

查看前端`app.js`中的收益率曲线绘制逻辑：
```javascript
// 检查这部分代码
function displayCumulativeReturnChart(data) {
  // 累计收益率计算逻辑
  // 确保使用正确的累乘方法
}
```
