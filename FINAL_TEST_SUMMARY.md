## 📖 使用指南

### 快速开始

1. **启动服务器**
   ```bash
   node server.js
   ```

2. **访问主页**
   ```
   http://localhost:3001/
   ```

3. **查看温度详情**
   - 点击"市场温度计"标签
   - 点击"查看详情 →"按钮

### API使用

```bash
# 获取当前综合温度
curl http://localhost:3001/api/composite-temperature

# 获取近3年多指数历史温度
curl "http://localhost:3001/api/multi-index-temperature?startDate=20230101&endDate=20260106"
```
