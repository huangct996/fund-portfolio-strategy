# 年化波动率和卡玛比率修复说明

## 修复时间
2026年1月13日 上午9:51

## 问题描述

用户反馈报告中存在以下问题：
1. **年化波动率显示为0.00%** - 所有策略的年化波动率都是0
2. **卡玛比率显示为0.00** - 所有策略的卡玛比率都是0
3. **年化波动率和卡玛比率的图表为空** - 柱状图没有数据显示

## 问题根源

### 字段名不匹配

**后端返回的字段名**：
```javascript
{
  volatility: 0.1466,  // 年化波动率
  // 没有 annualizedVolatility 字段
  // 没有 calmarRatio 字段
}
```

**报告脚本期望的字段名**：
```python
metrics = [
    ('年化波动率', 'annualizedVolatility', '%'),  # ❌ 字段名不匹配
    ('卡玛比率', 'calmarRatio', '')              # ❌ 字段不存在
]
```

### 缺失卡玛比率计算

后端风险指标计算中没有计算卡玛比率（Calmar Ratio）。

**卡玛比率定义**：
```
卡玛比率 = 年化收益率 / 最大回撤
```

用于衡量单位回撤下的收益能力，数值越高越好。

## 修复方案

### 1. 添加字段别名

在后端风险指标返回中，添加 `annualizedVolatility` 作为 `volatility` 的别名。

**修复文件**：`services/indexPortfolioService.js`

#### 修复位置1：基于每日收益率的风险指标计算（第1240-1260行）

## 卡玛比率解读

### 计算公式

```
卡玛比率 = 年化收益率 / 最大回撤
```

### 数值含义

| 卡玛比率 | 含义 | 评价 |
|---------|------|------|
| > 1.0 | 年化收益率 > 最大回撤 | 优秀 |
| 0.5 - 1.0 | 年化收益率 = 50%-100% 最大回撤 | 良好 |
| < 0.5 | 年化收益率 < 50% 最大回撤 | 一般 |

## 年化波动率解读

### 计算方法

```
年化波动率 = 日波动率 × √252
```

其中252是一年的交易日数。

### 数值含义

| 年化波动率 | 风险水平 | 适合投资者 |
|-----------|---------|-----------|
| < 10% | 低风险 | 保守型 |
| 10% - 20% | 中等风险 | 稳健型 |
| > 20% | 高风险 | 激进型 |

### 示例数据（2023年末至今）

**增强策略**：14.66% - 中等风险 ✅

**中证红利**：11.62% - 中等风险 ✅

**512890基金**：8.53% - 低风险 ✅

## 技术细节

### 为什么需要字段别名？

1. **向后兼容**：保持 `volatility` 字段，不影响现有代码
2. **前端兼容**：添加 `annualizedVolatility` 字段，满足报告脚本需求
3. **语义清晰**：`annualizedVolatility` 比 `volatility` 更明确

### 卡玛比率的优势

相比夏普比率：
- ✅ 更直观：直接对比收益和回撤
- ✅ 更实用：投资者更关心回撤而非波动率
- ✅ 更稳定：不受无风险利率变化影响
