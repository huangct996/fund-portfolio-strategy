# 任务完成总结

## 一、风险平价策略详细说明

### 1.1 核心逻辑

风险平价（Risk Parity）策略的核心思想是：**让每只股票对组合总风险的贡献相等**，而不是让每只股票的权重相等。

**基本原理**：
- 传统等权重策略中，高波动率股票会对组合风险贡献更多
- 风险平价通过给低波动率股票更高权重，给高波动率股票更低权重，来平衡风险贡献
- 权重公式：权重ᵢ ∝ 1 / 波动率ᵢ

### 1.2 计算流程

#### 步骤0：股票池筛选（可选）
如果启用股票池筛选，会先对股票进行以下筛选：

1. **ROE筛选**（可勾选启用）
   - 数据来源：Tushare `fina_indicator` 接口
   - 数据时间：最近一期财报
   - 筛选逻辑：剔除 ROE < 设定阈值的股票（如 ROE < 8%）
   - 实现方式：如果不勾选，minROE = 0，相当于不筛选

2. **负债率筛选**（可勾选启用）
   - 数据来源：Tushare `fina_indicator` 接口
   - 数据时间：最近一期财报
   - 筛选逻辑：剔除 负债率 > 设定阈值的股票（如负债率 > 60%）
   - 实现方式：如果不勾选，maxDebtRatio = 1.0（100%），相当于不筛选

3. **动量筛选**
   - 数据来源：股票历史价格
   - 数据时间：调仓日往前推N个月（默认6个月）
   - 计算方式：动量收益率 = (最后价格 - 第一价格) / 第一价格
   - 筛选逻辑：剔除动量收益率 < 设定阈值的股票（如 < -10%）

4. **质量得分筛选**
   - 数据来源：PE、PB（调仓日当天）
   - 计算方式：质量因子 = (1/PE + 1/PB) / 2
   - 筛选逻辑：剔除质量得分低于中位数的股票

#### 步骤1：获取历史价格数据
- **时间范围**：调仓日往前推 volatilityWindow 个月（默认12个月）
- **数据要求**：至少 volatilityWindow × 20 × 0.8 个交易日的数据（默认192个交易日）
- **价格类型**：使用复权后的收盘价（adj_close）
- **数据来源**：Tushare `daily` 接口 + `adj_factor` 接口

**示例**：
```
调仓日：2024-12-31
波动率窗口：12个月
数据起始日：2023-12-31
数据结束日：2024-12-31
```

#### 步骤2：计算每只股票的波动率（EWMA方法）

**EWMA（指数加权移动平均）公式**：
```
EWMA波动率 = √(EWMA方差)
EWMA方差ₜ = λ × EWMA方差ₜ₋₁ + (1-λ) × 收益率ₜ²
```

**参数**：
- λ（衰减因子）= 0.94（默认值）
- 收益率ₜ = (价格ₜ - 价格ₜ₋₁) / 价格ₜ₋₁

**EWMA的优势**：
- 对近期数据赋予更高权重（λ=0.94意味着94%权重给历史，6%给当前）
- 平滑处理，减少噪音影响
- 更能反映当前市场状况

#### 步骤3：计算基础风险平价权重

**标准风险平价**（默认方法）：
```
逆波动率ᵢ = 1 / 波动率ᵢ
权重ᵢ = 逆波动率ᵢ / Σ(逆波动率ⱼ)
```

**协方差矩阵优化**（可选，useCovariance=true）：
- 构建股票收益率的协方差矩阵
- 考虑股票之间的相关性
- 基于协方差矩阵计算风险平价权重

#### 步骤4：应用权重上限
```
权重ᵢ = min(权重ᵢ, maxWeight)
```
应用权重上限后重新归一化，确保权重总和为100%。

#### 步骤5：混合策略（可选）
如果启用混合策略（hybridRatio > 0）：
```
最终权重 = (1 - hybridRatio) × 风险平价权重 + hybridRatio × 市值权重
```

#### 步骤6：质量因子倾斜（可选）
如果启用质量因子倾斜（useQualityTilt=true）：
```
质量得分 = 股息率 × 10 + 1/PE
质量乘数 = 1 + min(质量得分 × 0.1, 0.3)
调整后权重 = 原权重 × 质量乘数
```

#### 步骤7：动量因子倾斜（可选）
如果启用动量因子倾斜（useMomentumTilt=true）：
```
动量收益率 = (最后价格 - 第一价格) / 第一价格
动量得分 = (动量收益率 - 最小动量) / (最大动量 - 最小动量)
动量乘数 = 1 + 动量得分 × momentumWeight
最终权重 = 风险平价权重 × 动量乘数
```

### 1.3 数据使用时间段总结

| 数据类型 | 时间范围 | 用途 | 数据来源 |
|---------|---------|------|---------|
| 股票日收益率 | 调仓日往前12个月 | 计算波动率（EWMA） | Tushare daily + adj_factor |
| 动量价格数据 | 调仓日往前6个月 | 计算动量收益率 | Tushare daily + adj_factor |
| ROE、负债率 | 最近一期财报 | 股票池筛选 | Tushare fina_indicator |
| PE、PB、股息率 | 调仓日当天 | 质量因子计算、股票池筛选 | Tushare daily_basic |
| 市值 | 调仓日当天 | 市值加权（混合策略） | Tushare daily_basic |

### 1.4 合理性分析

#### 优点：
1. **风险分散更均衡**：避免高波动率股票主导组合风险
2. **适应市场变化**：EWMA方法对近期数据赋予更高权重
3. **多因子增强**：可选的质量因子、动量因子倾斜
4. **风险控制**：单只股票最大权重限制、股票池筛选

#### 潜在问题：
1. **波动率窗口（12个月）**：市场环境快速变化时可能滞后
2. **EWMA衰减因子（0.94）**：固定值可能不适用所有市场环境
3. **财务数据滞后**：财报数据可能滞后3-6个月
4. **权重上限（10%）**：对于50只股票的组合可能偏高

#### 改进建议：
1. 可考虑6-18个月的动态窗口
2. 可测试0.90-0.97之间的不同衰减因子
3. 可根据股票数量动态调整权重上限

## 三、参数配置分析（最大化夏普比率）

### 3.1 测试结果分析

根据自动测试结果（2020-2024年回测）：

| 配置 | 夏普比率 | 年化收益 | 最大回撤 | 年化波动率 |
|-----|---------|---------|---------|-----------|
| 不启用ROE和负债率筛选 | **0.728** | 14.14% | 16.68% | 16.67% |
| 只启用ROE筛选 | 0.590 | 14.31% | 17.61% | 20.89% |
| 只启用负债率筛选 | 0.501 | 15.88% | 21.82% | 27.72% |
| 同时启用ROE和负债率筛选 | 0.437 | 15.44% | 26.47% | 30.78% |
| 推荐配置（窗口9月） | 0.436 | 15.42% | 26.48% | 30.79% |

### 3.2 关键发现

1. **最高夏普比率配置**：不启用ROE和负债率筛选
   - 夏普比率：0.728
   - 年化收益：14.14%
   - 最大回撤：16.68%
   - 原因：过度筛选会减少股票数量，增加集中度风险，导致波动率上升

2. **收益率vs风险的权衡**：
   - 启用更多筛选条件可以提高收益率（14.14% → 15.88%）
   - 但同时会显著增加波动率（16.67% → 27.72%）
   - 导致夏普比率下降（0.728 → 0.501）

3. **筛选条件的影响**：
   - ROE筛选：轻微增加收益，但增加波动率
   - 负债率筛选：显著增加收益，但大幅增加波动率
   - 同时启用：收益提升有限，但波动率大幅上升

### 3.3 最优参数配置建议

**基于夏普比率最大化**：

```javascript
{
  // 基础参数
  volatilityWindow: 12,          // 12个月窗口，平衡稳定性和灵活性
  ewmaDecay: 0.94,               // 标准衰减因子
  maxWeight: 0.10,               // 10%权重上限
  rebalanceFrequency: 'yearly',  // 年度调仓，降低交易成本
  
  // 不使用倾斜因子
  useQualityTilt: false,
  useCovariance: false,
  useMomentumTilt: false,
  hybridRatio: 0,
  
  // 适度的股票池筛选
  enableStockFilter: true,
  stockFilterParams: {
    minROE: 0,                   // 不启用ROE筛选
    maxDebtRatio: 1.0,           // 不启用负债率筛选
    momentumMonths: 6,
    minMomentumReturn: -0.10,    // -10%动量门槛
    filterByQuality: true        // 启用质量筛选
  }
}
```

**理由**：
1. 纯风险平价策略已经能很好地分散风险
2. 适度的质量筛选和动量筛选可以剔除明显的低质量股票
3. 不启用ROE和负债率筛选可以保持足够的股票数量，降低集中度风险
4. 避免过度优化和过拟合
5. 参数简单，易于理解和维护

### 3.4 不同风险偏好的配置

**保守型（低风险）**：
- 波动率窗口：12-18个月（更稳定）
- 不启用任何财务筛选
- 只启用质量筛选和适度的动量筛选
- 预期夏普比率：0.7-0.8

**平衡型（中等风险）**：
- 波动率窗口：9-12个月
- 可选择性启用ROE筛选（≥8%）
- 启用质量筛选和动量筛选
- 预期夏普比率：0.5-0.7

**进取型（高风险高收益）**：
- 波动率窗口：6-9个月（更灵活）
- 启用ROE和负债率筛选
- 可考虑启用动量因子倾斜
- 预期夏普比率：0.4-0.6，但收益率更高
