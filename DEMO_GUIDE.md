# 🌡️ 多指数综合温度计算功能 - 演示指南

## 🎯 核心改进

### 之前：单一指数
- 仅使用沪深300指数
- 无法反映中小盘市场情况

### 现在：多指数综合
- **沪深300**（35%）+ **中证500**（30%）+ **中证1000**（20%）+ **上证50**（15%）
- 全面反映市场整体估值水平

---

## 🚀 快速体验

### 1. 主页温度展示
访问：http://localhost:3001/

**操作步骤：**
1. 点击顶部"市场温度计"标签
2. 查看当前综合温度（71°，高估）
3. 点击"查看详情 →"按钮

### 2. 温度详情页面
访问：http://localhost:3001/temperature-detail.html

**功能展示：**
- ✅ 综合温度大卡片（渐变背景）
- ✅ 四个指数温度对比（网格布局）
- ✅ 历史温度曲线（可选择显示/隐藏各指数）
- ✅ 温度分布统计（低估/中估/高估占比）

### 3. API测试
```bash
# 当前综合温度
curl http://localhost:3001/api/composite-temperature | jq

# 近1年历史温度
curl "http://localhost:3001/api/multi-index-temperature?startDate=20240101&endDate=20250106" | jq
```
