# 风险平价策略参数优化建议

## 当前基准配置
```javascript
{
  volatilityWindow: 6,
  ewmaDecay: 0.91,
  rebalanceFrequency: 'quarterly',
  enableTradingCost: false,
  tradingCostRate: 0,
  riskFreeRate: 0.02,
  useQualityTilt: true,
  useCovariance: false,
  hybridRatio: 0.3,
  maxWeight: 0.13
}
```

**当前结果**：
- 夏普比率：0.40
- 年化收益：8.11%
- 年化波动：15.41%
- 最大回撤：13.59%

## 优化方向

### 方向1：降低市值权重混合比例（推荐优先测试）

**理论依据**：
- 当前 `hybridRatio: 0.3` 意味着30%市值权重 + 70%风险平价
- 市值权重会导致向大盘股倾斜，可能降低分散效果
- 纯风险平价（hybridRatio: 0）理论上应该有更好的风险调整后收益

**测试配置1：纯风险平价**
```javascript
{
  hybridRatio: 0,  // 100%风险平价
  其他参数保持不变
}
```

**测试配置2：轻度混合**
```javascript
{
  hybridRatio: 0.1,  // 10%市值权重
  其他参数保持不变
}
```

**测试配置3：中度混合**
```javascript
{
  hybridRatio: 0.2,  // 20%市值权重
  其他参数保持不变
}
```

### 方向2：调整波动率窗口期

**理论依据**：
- 当前 `volatilityWindow: 6` 使用6个月的历史数据
- 窗口太短：对短期波动过于敏感
- 窗口太长：对市场变化反应迟钝

**测试配置4：较长窗口**
```javascript
{
  volatilityWindow: 12,  // 12个月
  hybridRatio: 0,  // 配合纯风险平价
  其他参数保持不变
}
```

**测试配置5：较短窗口**
```javascript
{
  volatilityWindow: 3,  // 3个月
  hybridRatio: 0,
  其他参数保持不变
}
```

### 方向3：调整EWMA衰减系数

**理论依据**：
- 当前 `ewmaDecay: 0.91` 
- 较高的衰减系数（接近1）：更重视历史数据
- 较低的衰减系数：更重视近期数据

**测试配置6：更重视近期**
```javascript
{
  ewmaDecay: 0.85,  // 降低衰减系数
  hybridRatio: 0,
  其他参数保持不变
}
```

**测试配置7：更重视历史**
```javascript
{
  ewmaDecay: 0.95,  // 提高衰减系数
  hybridRatio: 0,
  其他参数保持不变
}
```

### 方向4：调整最大权重限制

**理论依据**：
- 当前 `maxWeight: 0.13` (13%)
- 较低的最大权重：更分散，但可能错过优质股票
- 较高的最大权重：允许集中持仓，但风险更高

**测试配置8：更分散**
```javascript
{
  maxWeight: 0.10,  // 10%
  hybridRatio: 0,
  其他参数保持不变
}
```

**测试配置9：允许适度集中**
```javascript
{
  maxWeight: 0.15,  // 15%
  hybridRatio: 0,
  其他参数保持不变
}
```

### 方向5：调整调仓频率

**理论依据**：
- 当前 `rebalanceFrequency: 'quarterly'` (季度)
- 更频繁调仓：更好地适应市场变化，但交易成本高
- 更低频调仓：交易成本低，但可能错过调整机会

**测试配置10：半年调仓**
```javascript
{
  rebalanceFrequency: 'semiannual',
  hybridRatio: 0,
  其他参数保持不变
}
```

**测试配置11：月度调仓**
```javascript
{
  rebalanceFrequency: 'monthly',
  hybridRatio: 0,
  其他参数保持不变
}
```

### 方向6：关闭质量因子倾斜

**理论依据**：
- 当前 `useQualityTilt: true` 向高股息、低PE股票倾斜
- 质量因子可能在某些市场环境下失效
- 纯风险平价不应该有因子倾斜

**测试配置12：纯风险平价（无质量倾斜）**
```javascript
{
  useQualityTilt: false,  // 关闭质量因子
  hybridRatio: 0,
  其他参数保持不变
}
```

## 快速测试方法

在网页界面中：

1. 保持其他参数不变
2. 只修改 `混合比例` 滑块：
   - 拖到最左边（0%）= 纯风险平价
   - 观察夏普比率变化

3. 如果夏普比率提升，继续测试：
   - 取消勾选 `启用质量因子倾斜`
   - 观察夏普比率变化

4. 找到最优配置后，再测试不同的：

   - 调仓频率（月度、季度、半年）

## 预期结果

基于理论分析，最优配置可能是：

```javascript
{
  volatilityWindow: 6,
  ewmaDecay: 0.91,
  rebalanceFrequency: 'quarterly',  // 或 'semiannual'
  enableTradingCost: false,
  tradingCostRate: 0,
  riskFreeRate: 0.02,
  useQualityTilt: false,  // 关闭
  useCovariance: false,   // 关闭
  hybridRatio: 0,         // 纯风险平价
  maxWeight: 0.13
}
```

**预期夏普比率**：0.45-0.55

## 为什么协方差优化会降低夏普比率？

可能的原因：

1. **样本量不足**：
   - 协方差矩阵需要大量历史数据
   - 当前只有6个月的窗口期，数据可能不够

2. **估计误差**：
   - 协方差矩阵的估计误差会被优化过程放大
   - 导致过度集中在某些股票上

3. **过拟合**：
   - 协方差优化可能过度拟合历史数据
   - 在样本外表现不佳

4. **实现问题**：
   - 可能需要更复杂的协方差估计方法
   - 如收缩估计器（Ledoit-Wolf）

**建议**：暂时不使用协方差优化，专注于简单有效的风险平价策略。

## 总结

**核心建议**：
1. ✅ 降低 `hybridRatio` 到 0-0.1（最重要）
2. ✅ 考虑关闭 `useQualityTilt`
3. ✅ 保持 `useCovariance: false`
4. ⚠️ 其他参数可以微调，但影响较小

**预期提升**：
- 夏普比率：0.40 → 0.45-0.55
- 波动率：可能略微上升
- 收益率：可能略微上升
